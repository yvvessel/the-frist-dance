#  The Frist Dance

Site do meu time **The Frist Dance** para exibir jogadores e ranking de Valorant.

##  Sobre o projeto

Este projeto é um website desenvolvido para mostrar os membros do time e destacar o melhor jogador (MVP), inspirado na tela de vitória do Valorant.

##  Funcionalidades

- Página inicial com identidade do time
- Página do time com todos os jogadores
- Destaque automático do melhor jogador (MVP)
- Layout inspirado no Valorant (players lado a lado)
- Estrutura preparada para integração com API

##  Como funciona

O projeto é dividido em:

### Frontend
- HTML (estrutura)
- CSS (estilo)
- JavaScript (lógica)

### Lógica
- Carrega jogadores
- Define ranking
- Identifica o MVP
- Centraliza o melhor jogador na tela
